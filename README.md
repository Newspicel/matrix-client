# Lattice

A desktop [Matrix](https://matrix.org) client for macOS and Linux.

Built with Electron, React, and the [matrix-js-sdk](https://github.com/matrix-org/matrix-js-sdk).

## Development

```bash
pnpm install
pnpm dev
```
